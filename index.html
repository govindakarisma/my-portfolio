Tentang Saya

Perkenalkan nama saya Govinda Kharisma Dewa, lulusan S1 Program Studi Sistem Informasi Fakultas Sains dan Teknologi UIN Suska Riau pada tahun 2021. Memiliki pengalaman bekerja sebagai Web Programmer selama satu tahun lebih di bidang Pengembangan dan Pengelolaan Aplikasi.

Dalam pengembangan aplikasi saya memiliki cukup pengalaman menggunakan bahasa pemrograman PHP dengan menerapkan framework Laravel dengan arsitektur Monolith serta juga dapat dalam membuat Rest API, Vector dan juga dapat menggunakan Git & GitHub.
